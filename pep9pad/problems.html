Problems

This section contains a list of test programs for the Chapter 8 problems from the text, Computer Systems, J. Stanley Warford, Fifth edition, Jones and Bartlett

See the section Writing Trap Handlers in this Help system for instructions on how to write a trap handler. The test drivers in this section correspond to Step 5 in those instructions.

To test a program first write the trap handler as described in Steps 1 through 4 of the instructions. For Step 5, you can simply click the Copy to Object button and the test program will be copied to the assembly language pane in the application. You can then run or trace the program in the application as usual.
