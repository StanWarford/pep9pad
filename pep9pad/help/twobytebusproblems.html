Two-byte Bus Problems

This section contains a list of microcode problems for the two-byte bus from the text, Computer Systems, J. Stanley Warford, Fifth edition, Jones and Bartlett, Publishers, 2017.

Each problem contains a documentation section followed by one or more unit tests described by the UnitPre and UnitPost specifications. To complete a problem click the Copy to Microcode button, and the documentation and unit test will be copied to the Microcode source pane in the application. You can then complete the problem by writing the microcode program as described in the Microcode Use section of this help system.
